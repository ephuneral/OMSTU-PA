Инструкции
Инструкции В генеалогическом древе определите для двух элементов их наименьшего общего предка (Lowest Common Ancestor). Наименьшим общим предком элементов A и B является такой элемент C, что С является предком A, C является предком B, при этом глубина C является наибольшей из возможных. При этом элемент считается своим собственным предком. Формат входных данных аналогичен предыдущей задаче Для каждого запроса выведите наименьшего общего предка данных элементов.
